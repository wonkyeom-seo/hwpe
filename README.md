# 한글 문서 웹 에디터

브라우저에서 실행되는 단일 파일 기반 에디터입니다.

## 실행 방법

```bash
python3 -m http.server 8080
```

브라우저에서 `http://localhost:8080` 접속 후 `index.html` 사용.

## 주요 기능

- 업로드: `.txt`, `.md`, `.html`, `.htm`, `.doc`, `.docx`, `.hwpx`, `.hwp`
- 편집: 굵게/기울임/밑줄/취소선, 정렬, 목록, 들여쓰기
- 서식: 폰트, 글자 크기, 글자색, 배경색
- 검색/치환: 찾기, 1회 치환, 전체 치환
- 삽입: 표, 현재 날짜
- 저장: HTML 저장, TXT 저장
- 자동 저장: 로컬 스토리지 임시저장 및 복원
- 인쇄/PDF 저장: 브라우저 인쇄 기능 사용

## 제한 사항

- `.hwp`는 브라우저 단독 환경에서 직접 파싱이 어려워 내용 편집을 지원하지 않습니다.
- `.doc/.docx`는 텍스트 파일처럼 로드되며, 복잡한 서식 보존은 지원하지 않습니다.
